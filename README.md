# Python_Project_Diwali_Sales_Analysis

“In this project, I analyzed Diwali sales data using Python to identify customer trends and improve business decisions.
First, I performed data cleaning by handling missing values and removing duplicates using Pandas. 
Then I conducted exploratory data analysis and created visualizations using Matplotlib and Seaborn.
I discovered that females contributed more to total sales, especially in the 26–35 age group. 
Married customers had higher purchasing power. 
The top-performing sectors included IT and healthcare, and popular product categories were clothing, food, and electronics.
From a business perspective, these insights help companies design targeted campaigns, focus on high-value customer segments, 
and optimize product offerings during festive seasons.”
“This project helped me strengthen my skills in data cleaning, analysis, and deriving actionable insights.”
